Project Summary
Schedulink is a web application that combines the functionalities of Google Classroom and Trello, allowing teachers and students to manage classes, assignments, and real-time laboratory monitoring. It features separate dashboards for teachers and students, enabling effective task management and collaboration.

Project Module Description
Teacher Features: Create classes, monitor student tasks, manage assignments with a Kanban board, and real-time lab monitoring.
Student Features: Join classes, complete assignments with various submission formats, and track progress using a Kanban board.
Lab Monitoring: Real-time status updates for computer laboratories, including QR code scanning to manage lab availability.
